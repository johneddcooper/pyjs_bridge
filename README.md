# pyjs_bridge
Allows running and interacting with Node JS runtimes using requests and an express API.

## Dependencies
python 3.*
requests *
pytest * (for testing only)
node 8.*
express 4.*

## Setup

`pipenv install`
`yarn install`

## Tests
`pipenv shell`
`pytest test_pyjsbridge.py`
